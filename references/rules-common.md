### G01｜违法犯罪与违禁活动（风险：极高）

- **适用**：所有内容，不限商业
- **怎么判**：跨全部素材判断：内容是否在实施、组织、帮助、招募、交易、推广或提供可执行的违法犯罪、赌博、违禁物品/活动信息？若为新闻、法律教育、风险提示或批判讨论，是否避免了可执行路径和招募推广？
- **不算违规**：新闻报道、法律教育、风险警示或明确批判，且不提供实施帮助
- **需要什么证据**：完整上下文与内容目的；涉及报道或教育时的来源、必要性及是否提供可执行细节
- **怎么改**：删除——删除交易、招募、推广、实施帮助和可执行步骤；改写——如属新闻或教育，明确批判/警示目的并最小化可模仿细节
- **判定后动作**：别发，核心内容本身就是问题
- **依据**：
  - 《抖音规则中心》（抖音，核验 2026-07-13）：“平台禁止制作、发布或传播扰乱经济秩序、妨碍社会管理秩序等侵害公共利益的违法内容或利用平台实施相关违法犯罪活动的行为，或发布为此类活动进行宣传、推广的内容。” [原文](https://www.douyin.com/rule/policy)
  - 《小红书社区规范》（小红书，核验 2026-07-13）：“2.3.1 禁止传播违法内容涉及违禁物品，包括但不限于：毒品、违禁药物、违法工具等，如大麻、鸦片、处方药物、投资理财产品、假币、枪支、烟草、爆炸物、翻墙软件、外挂程序等其他内容, b.涉及违禁活动，包括但不限于：有关传销、胎儿性别鉴定……” [原文](https://agree.xiaohongshu.com/h5/terms/ZXXY20221213003/-1)
  - 《微信视频号运营规范》（腾讯微信，核验 2026-07-13）：“5.1.7 散布淫秽、色情、赌博、暴力、恐怖或者教唆犯罪的；” [原文](https://weixin.qq.com/cgi-bin/readtemplate?lang=zh_CN&t=weixin_agreement&s=video&head=true)

### G02｜露骨色情与色情资源（风险：极高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否露骨描绘性行为或性部位，传播、贩卖、交换或导流色情资源？如属医学、生理、司法或教育内容，露骨呈现是否必要、克制并有清晰上下文？
- **不算违规**：必要的人体生理、医学、司法或教育内容，且最小化露骨呈现
- **需要什么证据**：画面、音频、文字及链接的完整内容；教育/医疗目的、受众和必要性说明
- **怎么改**：删除——删除露骨色情、色情资源和导流信息；改写——教育或医疗内容改为非露骨、必要且有伦理边界的表达
- **判定后动作**：别发，核心内容本身就是问题
- **依据**：
  - 《抖音规则中心》（抖音，核验 2026-07-13）：“平台禁止用户发布、传播含有色情、淫秽信息，或对色情、淫秽信息资源进行引流等违法违规内容。 色情、淫秽内容包括具体描绘性行为或者露骨宣扬色情的诲淫性的文字、图片、视频、音频等信息。制作、发布、传播、贩卖色情、淫秽内容属于违法行为，并可能……” [原文](https://www.douyin.com/rule/policy)

### G03｜性暗示与低俗博眼球（风险：高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否以性暗示、性挑逗、聚焦隐私部位、走光偷拍或庸俗标题吸引流量？教育、医疗或性别平等公共利益内容是否仍避免不必要的暗示、挑逗和不良联想？
- **不算违规**：教育、医疗或性别平等公共利益内容，遵守伦理且无挑逗性呈现
- **需要什么证据**：标题、封面、画面构图、动作、音效和上下文；公共利益或教育目的及最小化呈现说明
- **怎么改**：改写——删除挑逗性标题、台词、音效和暗示性表达；删除——裁切聚焦隐私部位、走光、偷拍或不必要裸露画面
- **判定后动作**：必须修改后才可发布
- **依据**：
  - 《抖音规则中心》（抖音，核验 2026-07-13）：“对于以博眼球、吸引流量为目的，宣扬、展示传播低级趣味，低俗、媚俗，具有性暗示或性挑逗意味的内容平台将采取相应的限制措施。” [原文](https://www.douyin.com/rule/policy)
  - 《小红书社区公约2.0》（小红书，核验 2026-07-13）：“社区明确禁止发布裸露或性暗示内容。如果你的分享是用于教育、医疗目的，或者性别平等公共利益，请严格遵守伦理规范，最大可能避免发布暗示、挑逗或引起不良联想的内容。” [原文](https://pgy.xiaohongshu.com/help/detail?id=1eda0a065dd894063c2e029a49e8f6a1&userType=4)
  - 《微信视频号运营规范》（腾讯微信，核验 2026-07-13）：“5.2.4 带有性暗示、性挑逗等易使人产生性联想的；” [原文](https://weixin.qq.com/cgi-bin/readtemplate?lang=zh_CN&t=weixin_agreement&s=video&head=true)

### G04｜未成年人性化与性侵害（风险：极高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否描绘、索取、传播、交易、美化、正常化或引导未成年人裸露、性行为、性暗示、性侵害，包含数字合成或软色情？保护性报道是否避免可识别受害者和不必要的侵害细节？
- **不算违规**：未成年人保护、司法新闻或预防教育，避免重现、性化或识别受害者
- **需要什么证据**：主体年龄与画面/文本完整上下文；涉及报道时的匿名化、必要性和二次伤害评估
- **怎么改**：删除——删除所有未成年人性化、裸露、性侵害重现及获取/传播路径；改写——保护性报道仅保留必要事实，彻底匿名并避免二次伤害
- **判定后动作**：别发，核心内容本身就是问题
- **依据**：
  - 《抖音规则中心》（抖音，核验 2026-07-13）：“平台禁止任何形式的描绘未成年人裸露或性行为的内容，包括以数字技术制作或合成的内容，以及未成年人相关的软色情内容。” [原文](https://www.douyin.com/rule/policy)
  - 《小红书社区规范》（小红书，核验 2026-07-13）：“2.2.1 涉及未成年人色情低俗的，如展示未成年人婚育、性侵害等” [原文](https://agree.xiaohongshu.com/h5/terms/ZXXY20221213003/-1)

### G05｜暴力威胁、煽动与血腥画面（风险：极高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否构成可信暴力威胁、组织煽动、美化暴力、提供暴力服务/工具说明，或不必要展示血腥残忍画面？新闻、纪录或公共认知内容是否适当打码、警示且不美化？
- **不算违规**：新闻、纪录、公共认知或反暴力教育，适当打码、警示且不美化
- **需要什么证据**：威胁对象、可信度、上下文和行为意图；血腥画面的必要性、遮挡、警示和新闻来源
- **怎么改**：删除——删除威胁、煽动、美化、暴力服务和工具说明；改写——新闻或教育内容最小化画面、打码并增加清晰风险警示
- **判定后动作**：别发，核心内容本身就是问题
- **依据**：
  - 《抖音规则中心》（抖音，核验 2026-07-13）：“平台坚决反对各种形式的暴力，禁止用户利用平台的服务煽动、宣扬或美化暴力，严禁用户利用平台服务组织或参与意图造成人员伤亡、重大财产损失、公共设施损坏、社会秩序混乱等严重的暴力活动。” [原文](https://www.douyin.com/rule/policy)
  - 《抖音规则中心》（抖音，核验 2026-07-13）：“平台允许为帮助人们提升对特定问题的认知或者媒体出于新闻报道的目的而发布血腥内容，但应进行适当打码处理。” [原文](https://www.douyin.com/rule/policy)
  - 《微信视频号运营规范》（腾讯微信，核验 2026-07-13）：“5.1.7 散布淫秽、色情、赌博、暴力、恐怖或者教唆犯罪的；” [原文](https://weixin.qq.com/cgi-bin/readtemplate?lang=zh_CN&t=weixin_agreement&s=video&head=true)

### G06｜自伤自杀（风险：极高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否展示、美化、鼓励、教唆、组织自伤自杀，提供策略方法，进行相关直播，或在讲述经历时缺乏必要风险提示？是否属于心理健康求助、科学研究、预防教育或康复叙事？
- **不算违规**：心理健康求助、科学研究、预防教育或康复叙事，避免方法细节并提供支持信息
- **需要什么证据**：完整叙事目的、方法细节和画面；风险提示、求助资源与是否可能强化模仿
- **怎么改**：删除——删除美化、鼓励、教唆、组织、方法和过程展示；改写——求助或预防内容避免方法细节，增加明确支持与危机干预信息
- **判定后动作**：别发，核心内容本身就是问题
- **依据**：
  - 《抖音规则中心》（抖音，核验 2026-07-13）：“平台欢迎用户发布心理健康相关的有益探讨和科学研究内容，但是禁止制作、发布或传播任何展示、宣扬、诱导、鼓励、组织或教唆自杀和自残的内容，或者任何泄漏自杀或自残受害者或幸存者身份并攻击他们的负面内容。禁止用户进行自杀或自残的直播。” [原文](https://www.douyin.com/rule/policy)
  - 《微信视频号运营规范》（腾讯微信，核验 2026-07-13）：“5.8.1 出现人或动物自虐、自残、自杀、被杀、吸食注射毒品或违禁药品致残以及枪击、刺伤、拷打等令人极度不适的内容;” [原文](https://weixin.qq.com/cgi-bin/readtemplate?lang=zh_CN&t=weixin_agreement&s=video&head=true)

### G07｜仇恨、歧视、侮辱与骚扰（风险：高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否基于性别、年龄、地域、民族、种族、宗教、国籍、身体/精神状况、受害经历、性别认同、经济或教育等身份特征贬低、排斥、威胁或煽动歧视，或针对个人实施侮辱、恐吓、组织攻击和持续骚扰？

- **需要什么证据**：目标个人/群体、身份特征、语境和发布者立场；引用内容的必要性与批判性上下文

### G08｜诈骗与欺骗性获利（风险：极高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否通过冒充、隐瞒、诱骗、虚假中奖/兼职/交友/募捐/投资等方式骗取信任、财物、互动，或将用户导向诈骗渠道？若为反诈曝光、受害者复盘或新闻，是否避免提供诈骗实施帮助？
- **不算违规**：反诈教育、受害者复盘或新闻报道，明确警示且不提供实施帮助
- **需要什么证据**：身份、承诺、收款/跳转路径和完整沟通上下文；反诈内容的来源、警示立场和细节最小化
- **怎么改**：删除——删除冒充、诱骗、收款、诈骗跳转和实施帮助；改写——反诈内容明确风险、删除可复用话术和有效诈骗入口

- **依据**：
  - 《抖音规则中心》（抖音，核验 2026-07-13）：“平台禁止用户利用平台服务，通过欺骗性手段骗取其他用户信任或将其他用户引导至其他站外平台进行诈骗；禁止用户通过各类欺骗性的行为或手段骗取互动和流量。” [原文](https://www.douyin.com/rule/policy)

### G09｜隐私与个人信息（风险：极高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否未经权利人许可且无法律规定的合理理由，公开、索取、交易或用于人肉搜索他人的身份、住址、联系方式、行程、财务、医疗、账号等个人信息或隐私？披露是否必要且已最小化？
- **不算违规**：本人发布、明确授权或有合法公开依据，且使用目的正当、披露最小化
- **需要什么证据**：权利人授权、合法依据、公开来源与使用目的；完整画面及不可逆打码/最小化披露说明
- **怎么改**：删除——删除身份、住址、联系方式、行程、财务、医疗和账号等信息；补证据——确需发布时补充明确授权或法律依据，并做不可逆脱敏
- **判定后动作**：别发，核心内容本身就是问题
- **依据**：
  - 《抖音规则中心》（抖音，核验 2026-07-13）：“平台禁止用户在未经权利人许可且无法律规定的合理理由的情况下发布、曝光或索取他人的个人信息或隐私，包括但不限于个人身份识别信息、财务信息、居住信息、医疗信息、线上或线下的活动信息以及其他不愿为他人所知晓的信息。” [原文](https://www.douyin.com/rule/policy)

### G10｜版权与未授权搬运（风险：高）

- **适用**：所有内容，不限商业
- **怎么判**：跨全部素材判断：发布者是否拥有原创权利、明确授权、许可、公共领域依据或适用的法定使用依据？是否构成抄袭、搬运或超出必要范围使用？仅注明来源是否被误当成授权？
- **不算违规**：原创、明确授权、许可、公共领域或有充分依据的必要引用
- **需要什么证据**：原创底稿、授权文件、许可范围或公共领域依据；引用目的、比例、必要性及来源标注
- **怎么改**：补证据——补充原创底稿、授权链、许可范围或法定使用依据；删除——无法证明权利基础时删除第三方素材或改用自有素材
- **判定后动作**：拿不准就先别发，补齐信息再判

### G11｜危险谣言与误导（风险：极高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否捏造、扭曲、断章取义、隐瞒真相或传播无事实基础的信息，并可能引发现实人身伤害、暴力、公共恐慌或高风险健康/食品安全误导？是否在求证、引用后辟谣或明确表达个人观点？
- **不算违规**：明确求证、引用后辟谣或可识别的个人观点，不把未证实内容冒充事实
- **需要什么证据**：原始来源、发布时间、完整上下文和权威核验材料；现实伤害路径及事实/观点区分
- **怎么改**：补证据——补充权威来源、时间和完整上下文；证据不足时不得下确定结论；改写——改为明确求证或辟谣表达，删除危险行动建议和恐慌性传播要求
- **判定后动作**：别发，核心内容本身就是问题

### G12｜未成年人权益与有害利用（风险：极高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否虐待、体罚、欺凌、利用或过度劳动未成年人，披露其隐私，诱导饮酒吸烟、危险模仿、打赏消费、厌学弃学等有害行为？正当宣教、新闻或权益保护是否避免二次伤害与身份暴露？
- **不算违规**：正当宣教、新闻或权益保护内容，匿名、克制且不诱导模仿
- **需要什么证据**：监护人同意、未成年人自愿与隐私保护材料；活动安全、劳动负荷、教育目的和二次伤害评估
- **怎么改**：删除——删除虐待、利用、危险行为、消费诱导和可识别隐私；改写——保护性内容匿名化、最小化呈现并增加安全/权益说明
- **判定后动作**：别发，核心内容本身就是问题

### G13｜危险行为与模仿（风险：极高）

- **适用**：所有内容，不限商业
- **怎么判**：内容是否展示、鼓励或指导在缺乏专业环境、技能、防护和警示下实施高危实验、挑战、驾驶、恶作剧、高空/明火行为，并可能诱发模仿？在视频号上，是否具体存在未成年人模仿风险？

- **需要什么证据**：专业资质、场地、防护、风险评估和警示；步骤可复制性、受众年龄与平台差异说明
- **怎么改**：删除——删除危险行为展示、挑战号召和可复制步骤；改写——必要教育改用结果说明或模拟，展示充分防护并增加醒目警示

### G14｜标题封面与正文严重不一致（风险：中）

- **适用**：所有内容，不限商业
- **怎么判**：跨标题、封面、首图、正文和视频判断：是否用标题/封面制造核心事实、结果或场景，但正文没有兑现或结论相反，达到严重不一致？是否只是合理概括或及时兑现的悬念？
- **不算违规**：合理概括、修辞或悬念表达，且正文及时、实质兑现
- **需要什么证据**：标题、封面、首图、正文和视频的跨素材对照；核心事实是否兑现及用户误导程度
- **怎么改**：改写——让标题和封面准确概括正文，不制造未兑现的核心事实；删除——删除与正文无关的猎奇封面、首图或标题元素
- **判定后动作**：必须修改后才可发布
- **依据**：
  - 《小红书社区规范》（小红书，核验 2026-07-13）：“4.1.3 发布严重影响用户体验的内容，如首图血腥恐怖、标题党、图文不符等” [原文](https://agree.xiaohongshu.com/h5/terms/ZXXY20221213003/-1)

# 抖音平台要点

在通用/商业/行业规则之上，发抖音时额外注意的增量。依据：抖音规则中心、巨量星图内容制作规范、短视频创作者管理规范、直播行为规范（核验 2026-07-13）。

## 抖音的独有严格点

- **商业内容尺度最细**：星图规范对绝对化用语、不可考证话术、收益保证、效果保证、专家名义、明星代言授权、促销字幕都有明文禁令——商业内容审核以 `rules-commercial.md` 全量过一遍，抖音是这些规则最直接的执法平台；
- **促销字幕**：视频画面/字幕里不得出现具体打折、满减、领券信息（例："满300减50"）——这条是抖音特有，其他平台没有对应明文；
- **专业资质门槛**：医疗、金融等专业内容要求"取得资质证照且经平台认证"后才能发布相关内容，无认证账号发专业结论直接踩线；
- **理财直播需平台事先认证**：未认证开展投资理财类直播（推荐、分析、预测股票基金等）明文禁止；
- **抽奖与一元购**：站内抽奖活动、"一元购/夺宝/众筹"话术明文禁止；
- **骗互动**：引导"点赞抽奖""评论区扣1领资料"等骗取互动的手段属于流量作弊类目。

## 发布页动作（检查单用）

- 命中即选标注：含 AI 生成内容 / 虚构演绎 / 营销信息 / 转载；AI 生成合成内容必须显式标识（监管要求，不是可选项）；
- 时事、社会事件类内容标注来源和日期；
- 商业合作走星图/官方渠道，私接广告不披露是隐性风险。

## 站外引流与第三方 app（星图预审直接驳回项）

依据抖音规则中心与巨量星图内容营销规范（[官方原文](https://www.douyin.com/rule/policy)）：引导脱离平台交易、诱导下载或使用第三方软件，是星图预审环节直接驳回的两类内容。判定要点：**这句话有没有给观众布置一个"平台之外"的动作**。

- 越线：给观众布置站外动作——下载、添加联系方式、扫码、去其他平台搜索。注意"让观众自己去应用商店搜"同样构成下载引导，动作由观众"自主"完成并不豁免；把站外动作设为领福利的前置条件（先下载/先注册才送）属于诱导加重项；画面中二维码、指向下载按钮的箭头等元素与口播同责；
- 不越线：只陈述自己的使用经历或产品事实、不布置动作的种草式介绍（官方问答明确允许此类表达）；商品自带的优惠信息属产品属性，本身不构成诱导；
- **同生态豁免**：剪映、抖音商城、抖店、巨量系工具、抖音团购与抖音同属一个生态，指向这些产品的引导不算站外外流；同一句"搜同款"，指向站内搜索没问题，指向其他电商平台就越线。

## 常见误区

- 抖音对"钱/赚钱"等普通表达不限流（副总裁公开辟谣过），不要建议用户改谐音；
- "第一/最好"在非商业的个人体验语境（"我今年第一次去"）不触发星图商业规则——先判商业属性再套规则。

# 修复手册

修复的目标是"安全地保住这条内容"，不是把它改得面目全非。改完用户认不出自己的稿子、钩子没了、数据没了，等于白改——他不会用你的版本。

## 修复阶梯（严格按序尝试）

1. **补披露/标注**：很多"违规"其实是"没标"。加一句"广告""与XX品牌合作""AI辅助生成""效果因人而异""个人观点，不构成投资建议"，内容一个字不用动。永远先检查这一层。
2. **补证据**：主张可证就留。"我月入3万"→"我月入3万（后台流水可查，口径为税前）"。给用户列清楚需要备什么证据（截图/报告/资质），发布前存好以备申诉。
3. **收缩主张**：把不可证的部分缩到可证范围。"全网最好用"→"我用过的同类产品里最顺手的"；"28天见效"→"我自己第四周开始有感觉，因人而异"。
4. **最小改写**：只重写越线的句子，句式和语气贴着原文。改写后信息量不许下降——删掉数字、案例、细节的改写是失败的改写。
5. **删除/放弃**：核心钩子本身就是违规锚点时（整条内容就是收益承诺），如实告知"这条内容的卖点不能这么说"，给出替代选题方向，不硬救。

## 三条铁律

- **禁止换皮**：把"减肥"改成"体重管理"但功效主张原样保留，不是修复，是给用户埋雷。改法必须真的降低实质风险。
- **禁止对抗**：谐音、拆字、拼音缩写、emoji 替代、文字遮挡，一律不给。平台对变体词单独打击，这是把小风险换成大风险。
- **保意优先**：开头钩子、个人语气、信息密度、合理的传播力是内容的命。`data/profile.md` 有改法偏好时严格遵循（比如"保留我的口头禅""给两个版本"）。

## 辟谣清单：这些"改法"不用做

用户常被民间谣言带偏做无谓的自我审查，遇到主动纠正（依据：平台官方澄清，见 scan.py 的辟谣提示）：

| 民间说法 | 事实 | 建议 |
|---|---|---|
| 说"钱/赚钱"会限流，要写"米/赚米" | 官方已澄清普通创作说"钱"不限流；谐音变体反而被单独关注 | 用正常表达；涉带货/理财时按商业规则审，而不是换谐音 |
| "微信"两个字不能出现 | 语境问题不是词的问题：提到微信不违规，"加微信成交"才是导流 | 删掉行动指令和联系方式，不用改称呼 |
| "死"字必须写成"S" | 正常叙事用字不触发规则；规避写法属于低质内容信号 | 正常书写 |

## 高风险选题的安全表达（对标法）

选题劲爆不等于必然违规——平台拦的多数是**表达方式**，不是话题本身（法律红线话题除外）。同一个劲爆话题，几乎总有合规爆款在正常获得流量，差别全在说法。改高风险选题时先问："这个话题的合规爆款是怎么表达的？"常用转换：

- 断言式钩子 → 过程记录式钩子（保住悬念，换掉断言）；
- 向观众的承诺 → 自己的经历+边界声明；
- 煽动情绪 → 给信息增量（数据、对比、亲测过程）；
- 神秘渠道暗示 → 公开来源引用。

改法参考优先级：`data/expressions.md` 里用户**实测**过的条目 > **规则可解释**的条目 > 本节通用策略；"仅观察"级条目不得用作改稿依据。

## 对标拆解（用户问"这条为什么没被封"）

用户拿别人的内容来问为什么安全时，用规则做归因，不用玄学：

1. 按正常流程对该内容做一遍判定，找出它**规避了哪些违规锚点**（如：只陈述经历没有承诺、有披露、非商业语境、行动指令指向站内）；
2. 能解释 → 把可迁移的手法写进 expressions.md（置信"规则可解释"），并向用户讲清"它安全靠的是什么，你抄这个思路时要保住哪个要件"；
3. 解释不通（看起来同样越线却没被处理）→ 如实说三种可能：幸存者偏差（还没被抽查）、账号差异（有资质/白名单/非商业）、执法延迟。记入 expressions.md 对标观察区标"仅观察"，**明确建议不要模仿**；
4. 别人"没被封"永远不能覆盖官方规则——规则明确禁止的表达，再多幸存案例也不改判。

## 风险语义框架的改写

有些表达单词全部无害，但叠加成"悬念恐吓+信源缺失"的框架就会拉高整条内容的风险评分（"内幕曝光""官方不敢说""马上删除""所有人都被骗了"）。改法是换框架不换观点：

| 高风险框架 | 低风险框架 |
|---|---|
| "没人敢告诉你的真相" | "很多人容易忽略的一点" |
| "官方终于承认了" | "根据公开资料显示" |
| "再不看就被删了" | 删除（人为紧迫感无信息量） |

## 平台变体

同一份内容各平台严格度不同时，先给"共同安全版"（按最严平台改），用户明确要求逐平台优化时再给平台变体。变体只在必要处不同，不做三份全文。

## 改完必须复检

修改稿重跑 scan + 判定，报告写清：已解决哪几条、是否引入新风险、还剩什么残余风险。跳过复检等于没审完。

# 我的沉淀

这个文件夹属于你，不属于工具。**这里的一切（除各 README 说明外）都不进 Git 仓库**——升级工具（`git pull` 或重装）物理上碰不到你的任何数据。全部文件都是普通文本，可用任何编辑器打开修改。

| 文件 | 是什么 |
|---|---|
| `profile.md` | 我的账号档案：常发平台、行业、商业模式、改法偏好。配一次，之后审核免问 |
| `my-rules.md` | 我的规则库：黑名单（帮我盯着的词）、语义规则、白名单、语义笔记。每次审核自动生效 |
| `expressions.md` | 我的安全表达库：改写对、换思路技巧、对标观察，带置信等级（实测/规则可解释/仅观察）。改稿时优先参考 |
| `wordlists/` | 大词库文件夹：一行一词的 `.md` 丢进来即生效 |
| `cases/` | 我的处罚案例复盘存档 |
| `history/` | 每次审核的报告存档 |

`profile.md`、`my-rules.md`、`expressions.md` 三个文件**首次使用时由 AI 从 `../templates/` 自动创建**；想自己动手，把 templates 里的同名文件复制过来即可。

不需要手动维护——对 AI 说"以后帮我盯着某个词""这是误报""这样说被拦了那样说就过了"，它会替你写好；问"我沉淀了什么"，AI 会读这里给你盘点。

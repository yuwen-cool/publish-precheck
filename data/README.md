# 我的沉淀

这个文件夹属于你，不属于工具。这里的内容（除本说明外）不进 Git 仓库，升级 Skill（`git pull`）不会动你的任何文件。全部文件都是普通文本，可用任何编辑器打开修改。

| 文件 | 是什么 |
|---|---|
| `profile.md` | 我的账号档案：常发平台、行业、商业模式、改法偏好。配一次，之后审核免问 |
| `my-rules.md` | 我的规则库：黑名单（帮我盯着的词）、语义规则、白名单（我场景里安全的词）、语义笔记。每次审核自动生效 |
| `expressions.md` | 我的安全表达库：改写对（这样说不行那样说可以）、换思路技巧、对标观察，全部带置信等级（实测/规则可解释/仅观察）。改稿时优先参考 |
| `wordlists/` | 大词库文件夹：一行一词的 `.md` 丢进来即生效（`词\|原因` 可附说明，`#` 开头是注释；格式见里面的示例文件，`.txt` 也兼容） |
| `cases/` | 我的处罚案例复盘存档 |
| `history/` | 每次审核的报告存档 |

不需要手动维护——对 AI 说"以后帮我盯着某个词""这是误报""把这份词表导进来"，它会替你写好；问"我沉淀了什么"，AI 会读这里给你盘点。首次使用时 AI 会按下面的模板帮你创建这两个文件。

## profile.md 模板

```markdown
# 我的档案

- 常发平台：（抖音 / 小红书 / 微信视频号，可多选）
- 内容领域：（如：AI 工具教学、职场成长、美食探店）
- 是否涉及强监管行业：（无 / 医疗健康 / 金融理财）
- 商业模式：（无变现 / 带货 / 卖课 / 接广告 / 咨询服务 / 引流私域）
- 我有的资质：（如：无 / 执业医师 / 基金从业资格）

## 改法偏好

- 保留我的语气和开头钩子，别改成官样文章
- 必改项直接给可替换的句子，不要只说"建议调整"
```

## my-rules.md 模板

```markdown
# 我的规则库

## 黑名单（有明确词面的，scan 自动盯防）

| 词/正则 | 为什么 | 平台 | 级别 | 判法 |
|---|---|---|---|---|

## 语义规则（没有固定词面、按意思判的个人规则）

| 编号 | 什么情况算命中（锚点写具体） | 怎么改 | 平台 | 级别 |
|---|---|---|---|---|

## 白名单（这些词在我的场景里是安全的，降低误报）

| 词 | 情境说明 |
|---|---|

## 语义笔记（背景知识：AI 判定时参考的我的语境说明）

<!-- 例：我的行业里"账号诊断"是正常服务名称，与医疗无关。 -->
```

黑名单"判法"两个取值：**语境判**（默认，命中后 AI 结合上下文判断，防误报）、**见词即报**（你确认过零容忍的词，命中直接进必改）。语义规则编号从 MY-01 起，AI 审核时和官方规则同等对待。

# 我的处罚案例库

每次被限流、删除、警告或封禁后的复盘存档放这里，一次一个文件，命名 `YYYY-MM-DD-一句话描述.md`。不用手写——对 AI 说"我被限流了，帮我复盘"，它复盘完会按下面的模板替你存档。

```markdown
# 案例：<一句话描述>
- 日期/平台/处罚类型：
- 平台通知原文：
- 内容关键片段（涉及隐私可脱敏）：
- 归因：A 可证实违规 / B 披露标注问题 / C 推荐质量 / D 未知信号 + 分析
- 命中规则：<规则编号或"未知">
- 学到的教训：
- 已固化：<写进 my-rules.md 的条目，或"无可固化项">
```

存档的价值：教训会同步固化进 `../my-rules.md`，以后每次审核自动带上；案例本身留档，下次遇到类似处罚可对照。
